# ReactJs_Docs_Comics

[![Live site](https://img.shields.io/badge/Live%20Site-GitHub%20Pages-blue?logo=github)](https://41chaitanya.github.io/ReactJs_Docs_Comics/)

A cool comic-themed React JS docs website for Sheryians and learners.

## About

This project is a fun, comic-style documentation site built as a static HTML project.
It includes a homepage and a dedicated `React_Documentation.html` page with React details.

## Live site

https://41chaitanya.github.io/ReactJs_Docs_Comics/

## Deployment

This repo uses GitHub Actions to deploy the site automatically on every push to `main`.

- Workflow: `.github/workflows/deploy.yml`
- Pages source: `gh-pages` branch

## Files

- `index.html` — home page
- `React_Documentation.html` — documentation page
- `.nojekyll` — disables Jekyll on GitHub Pages
